# Посмотри в окно
____
## Проектная работа второго спринта на курсе fullstack разработчик от Яндекс Практикум

### Описание проекта

Проект представляет из себя страницу со списком карточек городов. Переключаясь между карточками, можно просматривать видеофрагменты из разных городов мира.

Основная задача проекта - стилизация страницы с готовым html + js кодом.

### Проблематика проекта

В проекте можно выделить следующие группы решаемых в процессе разработки задач:

- Стилизация форм
- Кастомизация объектов форм
- Использование псевдоклассов и псевдоэллементов
- flex позиционирование объектов
- Стилизация состояний focus у объектов

### Функциональность

Для корректного отображения страницы рекомендуется использование актуальных, поддерживаемых версий браузеров.

### Дальнейшее развитие

- Доработка проекта после ревью
- Более глубокая проработка корректного отображения кастомных состояний focus объектов в разных браузерах
